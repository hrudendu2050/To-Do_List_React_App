# To-Do_List_React_App
This is a simple To-Do List application. The application allow users to add, delete, and mark tasks as completed.  It uses React functional components, state, and props to manage the application.
# Documentation
* Here 2 components are used- ToDoList- To add new tasks in the list
                              ToDoItem- To mark tasks(or items) completed, edit and delete the task
* The use of useState hook is done for component rerendering purposes
* Props are used to pass function as arguments, handle onClick and access task data within the list across components

# Access Application
* In Github, application is present in the master branch.
* GitHub link: https://github.com/hrudendu2050/To-Do_List_React_App.git
